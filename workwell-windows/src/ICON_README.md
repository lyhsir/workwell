# 图标文件说明

需要添加 `WorkWell.ico` 文件到 `src/` 目录。

## 生成图标的方法

### 方法 1：在线转换
访问 https://convertico.com/ 或类似网站，上传 PNG 图片转换为 ICO。

### 方法 2：使用 ImageMagick
```bash
magick icon.png -define icon:auto-resize=256,128,96,64,48,32,16 WorkWell.ico
```

### 方法 3：使用 GIMP
1. 打开 PNG 图片
2. 导出为 .ico 格式
3. 选择多个尺寸（16x16, 32x32, 48x48, 256x256）

## 临时方案

如果没有图标文件，代码会使用系统默认图标（IDI_APPLICATION）。
应用仍然可以正常运行，只是托盘图标会显示为默认的应用程序图标。

## 图标尺寸建议

- 16x16 - 系统托盘小图标
- 32x32 - 系统托盘大图标
- 48x48 - 标准图标
- 256x256 - 高清显示

建议使用 256x256 或更大的源图，然后包含多个尺寸。
