# Timeout 快速使用指南

## 🚀 快速开始

### 第一次运行
```bash
# 方式 1：使用启动脚本（推荐）
./run.sh

# 方式 2：直接运行
swift run Timeout
```

### 日常使用
应用启动后，会在菜单栏显示 🍅 图标。

#### 基本操作流程
1. **点击菜单栏的 🍅 图标** 打开控制面板
2. **点击"开始工作"** 开始 25 分钟倒计时
3. 工作时间结束后，**自动进入休息模式**
4. 休息期间屏幕会被遮罩，显示休息建议
5. 休息时间结束后，**自动恢复工作计时**

## ⌨️ 常用命令

### 运行应用
```bash
swift run Timeout
```

### 停止应用
```bash
pkill -f Timeout
```

### 重新构建
```bash
swift build
```

### 生成 Xcode 项目（可选）
```bash
swift package generate-xcodeproj
open Timeout.xcodeproj
```

## 🎯 使用技巧

### 自定义时间设置
1. 点击菜单栏图标
2. 点击右上角的 ⚙️ 齿轮图标
3. 调整工作时长、短休息、长休息等设置

### 选择强制休息强度

#### 🟢 轻度
- 适合刚开始使用的用户
- 全屏提示 + 声音提醒
- 显示呼吸练习和休息建议
- 可以通过 Cmd+Q 或点击关闭按钮退出

#### 🟡 中度（推荐）
- 适合需要强制休息的用户
- 半透明黑色遮罩
- 拦截常见快捷键（Cmd+W、Cmd+Q、Esc）
- 需要等待或认真对待休息时间

#### 🔴 重度
- 适合自制力较差的用户
- 不透明黑色遮罩
- 更强的限制
- 基本无法忽略休息时间

### 配置跳过休息选项

在设置中可以：
- **允许跳过休息**：开启/关闭跳过功能
- **跳过需要输入原因**：强制记录跳过理由
- **跳过延迟**：设置 0-30 秒延迟，防止误触

## 📊 了解统计数据

应用会追踪：
- **今日番茄数**：今天完成的番茄钟数量
- **专注时长**：今天的总专注时间
- **连续天数**：连续使用天数

## ⚠️ 注意事项

### 如果应用卡住
1. 按 `Cmd+Opt+Esc` 打开强制退出菜单
2. 选择 Timeout 并强制退出
3. 或者在终端运行：`pkill -9 Timeout`

### 如果休息窗口无法关闭
1. 等待休息时间结束（推荐）
2. 如果紧急情况，使用 Activity Monitor 强制退出

### 电脑睡眠期间
- 计时器会暂停
- 唤醒后继续计时
- 总时间会相应延长

## 🔧 故障排除

### 应用不显示在菜单栏
- 检查是否已在运行：`ps aux | grep Timeout`
- 重启应用：先 `pkill -f Timeout`，然后重新运行

### 计时不准确
- 避免让电脑进入睡眠状态
- 在系统设置中调整睡眠设置

### 找不到图标
- 检查菜单栏右侧是否有 🍅 图标
- 图标可能在菜单栏的隐藏区域

## 💡 最佳实践

1. **从轻度开始**：先使用轻度模式，逐渐适应后再增加强度
2. **真实记录**：如果要跳过休息，认真填写原因
3. **配合实际**：根据个人习惯调整工作/休息时长
4. **定期回顾**：查看统计数据，了解自己的工作模式
5. **保持连续**：尝试每天使用，建立习惯

## 🎓 工作法建议

### 番茄工作法基础流程
1. 选择一个任务
2. 设置 25 分钟计时器
3. 专注工作直到计时器响起
4. 短休息 5 分钟
5. 重复 4 次后，长休息 15-30 分钟

### 自定义建议
- **创意工作**：50/10 模式（50 分钟工作，10 分钟休息）
- **学习时间**：25/5 模式（经典番茄钟）
- **高强度任务**：15/3 模式（更频繁的休息）
